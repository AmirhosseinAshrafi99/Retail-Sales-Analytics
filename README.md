# Retail Sales Analytics

## Objective

Analyze retail sales performance and identify the factors affecting profitability.

## Tools

- Python
- Pandas
- Matplotlib
- Git
- GitHub
- Power BI

## Key Findings

- Technology generated the highest revenue and profit.
- Furniture generated strong sales but the lowest profit margin.
- Tables produced over $757K in revenue while generating a loss of $64K.
- Table profitability becomes negative when discounts exceed approximately 30%.
- Turkey and Nigeria were the largest loss-making countries.
- Average country discount rates showed a strong negative relationship with profit margin.
